Since 0.7.0.

権限レベル: GameDirectors.

プレイヤーに映像を見せるコマンド。

| Parameter | Required | Type           | Description      |
| --------- | :------: | -------------- | ---------------- |
| target    |    ◯     | PlayerSelector | 映像を見せる対象 |
| moviename |    ◯     | String         | 映像名           |

### Examples

```
/nacht:showcameramovie @s xxxxx
```
