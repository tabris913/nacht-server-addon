Since 0.4.0.

権限レベル: Any.

ダイスロールをおこなう。

| Parameter | Required | Type   | Description      |
| --------- | :------: | ------ | ---------------- |
| dice      |    ◯     | String | ダイスロール指示 |

### Examples

```
/nacht:diceroll 1D100
[playerName] 1D100 = 96 ( 96 )
```

```
/nacht:diceroll 2d6
[playerName] 2d6 = 7 ( 3 4 )
```
