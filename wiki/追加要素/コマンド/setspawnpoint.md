Since 0.6.0.

権限レベル: GameDirectors.

スポーン地点を設定するコマンド。

| Parameter | Required | Type           | Description  |
| --------- | :------: | -------------- | ------------ |
| target    |    ◯     | PlayerSelector | プレイヤー   |
| location  |    ◯     | Location       | スポーン地点 |

### Examples

```
/nacht:setspawnpoint @p 0 0 0
```
