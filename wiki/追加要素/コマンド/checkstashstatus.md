Since 1.1.0.

| Parameter | Required | Type           | Description        |
| --------- | :------: | -------------- | ------------------ |
| target    |    ◯     | PlayerSelector | 確認するプレイヤー |

### Examples

```
/nacht:checkstashstatus @initiator
```
