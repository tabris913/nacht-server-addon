Since 1.0.0.

権限レベル: GameDirectors.

プレイヤーの表示名を変更する。

| Parameter   | Required | Type           | Description        |
| ----------- | :------: | -------------- | ------------------ |
| target      |    ◯     | PlayerSelector | 変更するプレイヤー |
| displayName |    ◯     | String         | 表示名             |

### Examples

```
/nacht:setname xxxxxxx プレイヤーA
```
